# Credentials

This directory will be populated by the script 'scripts/create-credentials.py' that
will put here, for each Identity Provider Shibboleth, the following credentials files:

 * idp-backchannel.crt
 * idp-backchannel.p12
 * idp-encryption.crt
 * idp-encryption.key
 * idp-signing.crt
 * idp-signing.key
 * sealer.jks
 * sealer.kver
